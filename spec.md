# Single product page improvement

We need to do some additional work on the product page. It's important that we extend the default theme components, which are already on that page, instead of writing our own.

When extending functionality, each functionality should have a admin option, in order to customize it, unless otherwise stated.

## Add to cart button

The add to cart button should be always visible, on the page, on mobile. It should be pinned to the bottom of the viewport, and scrollable as the user scrolls. But, only when he scrolls a little bit from the header. It should show/hide.

This functionality should be toggleable on the admin.

## Price and name on top

The price and name of the product should come first, then the rest of the information. The current issue with the component is that this isn't allowed.

## Breadcrumb

we need a basic breadcrumb element, that we can add here, through the admin, so that the user can navigate.
